## 1. 문제 요약

`final-prompt-package`는 고객 화면의 주 브랜드를 `OPTIC`으로 정렬하고, 실제 서비스 진입 CTA를 더 명확히 보여주는 것을 첫 개선 단계로 제안한다. 현재 랜딩은 기존 브랜드 표기, hero 맥락, 서비스 링크가 섞여 있어 방문자가 어떤 이름과 경로를 기억해야 하는지 즉시 판단하기 어렵다.

F1은 이후 카피 정리와 신규 업무 섹션의 기준점이 되는 최소 브랜드 전환 작업이다. 범위는 header, footer, 로고, 상수, 모바일 메뉴 CTA처럼 작은 표면에 제한한다.

## 2. 목표

- 고객 화면에서 주 브랜드가 `OPTIC`으로 보이도록 header/footer/CTA 표기를 정리한다.
- `OPTICS`는 내부 엔진 또는 footer/About 수준의 보조 표기로 제한한다.
- `OPTIC 바로가기` CTA를 실제 서비스 URL `https://mm-broker-test.vercel.app/`로 연결한다.
- desktop header와 mobile menu에서 서비스 CTA와 문의 CTA가 구분되게 만든다.
- 로고 적용은 production 승인 전까지 임시 자산 또는 기존 `OpticLogo` 개선안으로 분리한다.

## 3. 예상 범위

| 영역 | 예상 작업 |
|---|---|
| Brand constants | `src/lib/constants.ts`의 브랜드명, CTA label, 서비스 URL 정리 |
| Header | `src/components/sections/header.tsx`에 `OPTIC 바로가기` CTA 추가 또는 정렬 |
| Footer | `src/components/sections/footer.tsx`에서 `OPTIC/OPTICS` 역할 분리 |
| Logo | `src/components/icons/optic-logo.tsx`의 표시 기준 확인 |
| Mobile menu | 모바일 메뉴에서 서비스 CTA와 문의 CTA 구분 |
