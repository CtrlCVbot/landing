# F5 UI Spec

## Header

- 좌측 브랜드는 `OpticLogo` 컴포넌트를 사용한다.
- 홈 링크의 접근 가능한 이름은 `OPTIC 로고`다.
- 서비스 CTA는 새 창 이동 라벨을 가진다.

## Footer

- 브랜드 영역은 `OpticLogo` 컴포넌트를 사용한다.
- 외부 제품 링크는 새 창으로 열며 보안용 `rel`을 가진다.

## Brand Assets

- `optic-logo.svg`: 공유 가능한 wordmark
- `optic-mark.svg`: 작은 아이콘과 favicon 기준
- `optic-og.svg`: Open Graph 공유 이미지
- `favicon.svg`: 브라우저 favicon
