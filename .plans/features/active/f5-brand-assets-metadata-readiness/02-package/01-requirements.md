# F5 Requirements

| ID | Requirement | Acceptance |
|---|---|---|
| R-F5-01 | 브랜드 자산 기준 추가 | `BRAND_ASSETS`가 logo, mark, favicon, Open Graph 경로를 제공 |
| R-F5-02 | metadata 정리 | `layout.tsx` metadata가 `SITE_METADATA`와 브랜드 자산을 참조 |
| R-F5-03 | 로고 접근성 | header/footer 로고가 `OPTIC 로고` 접근성 이름을 가짐 |
| R-F5-04 | 외부 링크 속성 | 새 창 링크에 `target="_blank"`와 `rel="noopener noreferrer"` 적용 |
| R-F5-05 | 검증 기록 | typecheck/test/lint/build/문구 스캔/preview 결과를 남김 |
