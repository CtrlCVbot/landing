# F4 UI Spec

## Workflow Status Board

- 위치: workflow step list 위에 작은 안내 band 1개
- 카드 내부: `summary`, `state title`, `events`
- 색상: progress/success/warning/neutral tone
- 모바일: status event chip은 1열로 wrap

## Motion

- list: stagger children
- card: opacity + y 16px reveal
- reduced motion: initial animation props disabled
