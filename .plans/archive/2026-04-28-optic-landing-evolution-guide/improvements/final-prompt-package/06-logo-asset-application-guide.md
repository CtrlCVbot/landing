# 06. 로고 자산 적용 가이드

## 1. 기준 이미지

| 항목 | 내용 |
| --- | --- |
| 기준 이미지 | `C:/Users/user/Downloads/ChatGPT Image 2026년 4월 28일 오후 03_23_39.png` |
| 사용 성격 | OPTIC 랜딩 로고 작업의 기준 이미지 후보 |
| 현재 상태 | 흰 배경 위 검정 `OPTIC` 워드마크 이미지 |
| 적용 범위 | 헤더 로고, footer 로고, favicon/app icon 후보, Open Graph 이미지의 브랜드 요소 |

이 이미지는 현재 랜딩 개선 문서에서 **OPTIC 고객 접점 브랜드 로고의 시각 기준**으로 사용한다.
단, 바로 production 자산으로 쓰기보다 투명 배경, SVG 또는 고해상도 PNG, dark mode 대응 자산으로 정리한 뒤 적용하는 것을 권장한다.

## 2. 시각 해석

| 요소 | 해석 | 랜딩 적용 의미 |
| --- | --- | --- |
| 원형 심볼 | 렌즈, 시야, 초점, 관제의 인상 | “운송 운영을 선명하게 보이게 한다”는 OPTIC 브랜드 메시지와 잘 맞음 |
| 세로 분할선 | 흐름을 구분하고 정렬하는 느낌 | 오더, 운송, 정산 흐름을 정리하는 제품 성격과 연결 가능 |
| 굵은 대문자 워드마크 | 안정감, B2B SaaS, 업무 도구 인상 | 과장된 장식보다 신뢰 중심의 랜딩 톤에 적합 |
| 검정 단색 | 높은 대비, 명확한 인식 | 밝은 배경 헤더와 문서형 랜딩에 우선 적용하기 좋음 |

브랜드 메시지는 `See logistics clearly.`, `오더부터 정산까지 한눈에.`와 연결한다.

## 3. 로고 표기 원칙

| 항목 | 기준 |
| --- | --- |
| 로고 텍스트 | `OPTIC`만 사용 |
| 금지 표기 | `Optic Cargo`, `OPTICS` 메인 로고 사용 |
| `OPTICS` 사용 위치 | footer/About의 `Powered by OPTICS` 보조 문구로 제한 |
| 제품 라인업 조합 | 필요할 때만 `OPTIC Broker`, `OPTIC Shipper`처럼 텍스트로 보조 |
| 한글 병기 | 로고 자체에는 넣지 않고 주변 문구에서만 `옵틱` 사용 가능 |

고객이 처음 보는 랜딩에서는 `OPTIC` 로고가 제품 브랜드로 먼저 인식되어야 한다.
`OPTICS`는 엔진 브랜드이므로 메인 로고나 헤더 로고로 쓰지 않는다.

## 4. 구현 시 필요한 파생 자산

| 자산 | 권장 형식 | 용도 |
| --- | --- | --- |
| Wordmark dark-on-light | `svg`, `png` | 밝은 배경 헤더, footer |
| Wordmark light-on-dark | `svg`, `png` | 어두운 섹션, dark background |
| Symbol only | `svg`, `png` | favicon, 모바일 좁은 헤더, app icon 후보 |
| Favicon | `svg`, `ico`, `png` | 브라우저 탭 |
| Apple touch icon | `png`, 180x180 이상 | 모바일 홈 화면 저장 |
| Open Graph image | `png`, 1200x630 | 공유 미리보기 |

구현 단계에서는 이미지 원본을 그대로 배치하기보다, 로고 가장자리와 글자가 흐려지지 않도록 SVG 벡터 또는 충분한 해상도의 투명 PNG로 정리한다.

## 5. 랜딩 배치 기준

| 위치 | 적용 방식 | 주의점 |
| --- | --- | --- |
| 상단 헤더 | 심볼 + `OPTIC` 워드마크 | 높이는 28~36px 범위에서 시작, CTA보다 과하게 크지 않게 유지 |
| 모바일 헤더 | 심볼 단독 또는 축약 워드마크 | 작은 화면에서 글자가 뭉개지면 심볼 단독 사용 가능 |
| hero | 현재 AI 오더 등록 애니메이션은 유지 | 로고를 hero 중심 요소로 키우지 않음 |
| footer | 작은 워드마크 + `Powered by OPTICS` | `OPTIC`과 `OPTICS` 역할 차이가 보이도록 위치 분리 |
| Open Graph | 로고 + 핵심 문구 | `오더부터 정산까지 한눈에.`를 함께 사용 가능 |

현재 hero 섹션은 유지 대상이므로, 로고는 브랜드 인지와 신뢰를 보강하는 역할로만 사용한다.

## 6. 접근성 및 품질 기준

| 항목 | 기준 |
| --- | --- |
| `alt` | `OPTIC` |
| 홈 링크 `aria-label` | `OPTIC 홈` |
| 실제 서비스 CTA와 구분 | 로고 클릭은 홈, `OPTIC 바로가기`는 서비스 이동 |
| 대비 | 밝은 배경에서는 검정 로고, 어두운 배경에서는 흰색 로고 사용 |
| 최소 크기 | favicon을 제외하고 워드마크 글자가 읽히는 크기 유지 |
| 여백 | 심볼과 글자 주변에 최소 로고 높이의 25% 이상 clear space 확보 |

로고 클릭과 `OPTIC 바로가기` 버튼은 목적이 다르다.
로고는 랜딩 홈으로 이동하고, CTA는 `https://mm-broker-test.vercel.app/` 실제 서비스로 이동한다.
